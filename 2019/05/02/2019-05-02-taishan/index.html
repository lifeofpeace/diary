泰山

不知道从何时起有了一个想法，每年的生日登一座名山，便是从五岳开始，五岳便从泰山开始。

时间回到六年前，再去回想有些记忆都模糊了，图片也只有QQ空间里存的那几张，没有找到手机拍的其他图片了。

2013年05月21日 k51 北京 - 泰山
2013年05月22日 k46 泰山 - 北京

可以说这两个时间点都是非常好的了，晚上卧铺睡一觉，第二天一早就可以登山了，当天晚上返程，第二天不耽误上班。

我查了一下5月22日是周三，工作日，难道我是请假去的吗，当天还在上班，晚上坐火车去爬了个泰山，第二天回来照常上班，像极了梁朝伟坐飞机去伦敦喂鸽子，O(∩_∩)O。我是真的不记得我为什么这么做了。
